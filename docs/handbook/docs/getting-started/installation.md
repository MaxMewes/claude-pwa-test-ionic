# Installation

Die labGate PWA kann auf verschiedenen Geräten und Plattformen installiert werden.

## Web-Browser

Die einfachste Möglichkeit, labGate zu nutzen, ist direkt im Webbrowser:

1. Öffnen Sie Ihren bevorzugten Browser (Chrome, Firefox, Safari, Edge)
2. Navigieren Sie zu `https://app.labgate.net`
3. Melden Sie sich mit Ihren Zugangsdaten an

## Als App installieren (PWA)

### Android

1. Öffnen Sie Chrome und navigieren Sie zu `https://app.labgate.net`
2. Tippen Sie auf das Menü-Symbol (drei Punkte) oben rechts
3. Wählen Sie "Zum Startbildschirm hinzufügen"
4. Bestätigen Sie mit "Hinzufügen"

### iOS (iPhone/iPad)

1. Öffnen Sie Safari und navigieren Sie zu `https://app.labgate.net`
2. Tippen Sie auf das Teilen-Symbol (Quadrat mit Pfeil nach oben)
3. Scrollen Sie nach unten und wählen Sie "Zum Home-Bildschirm"
4. Bestätigen Sie mit "Hinzufügen"

### Desktop (Windows/Mac)

=== "Chrome"

    1. Navigieren Sie zu `https://app.labgate.net`
    2. Klicken Sie auf das Installations-Symbol in der Adressleiste
    3. Bestätigen Sie die Installation

=== "Edge"

    1. Navigieren Sie zu `https://app.labgate.net`
    2. Klicken Sie auf die drei Punkte oben rechts
    3. Wählen Sie "Apps" → "Diese Seite als App installieren"

## Systemanforderungen

| Plattform | Mindestanforderung |
|-----------|-------------------|
| Android | Android 8.0+ mit Chrome |
| iOS | iOS 12+ mit Safari |
| Desktop | Moderner Browser (Chrome, Firefox, Edge, Safari) |

## Offline-Nutzung

Nach der Installation als PWA können Sie auf bereits geladene Befunde auch ohne Internetverbindung zugreifen. Neue Daten werden automatisch synchronisiert, sobald Sie wieder online sind.

!!! warning "Hinweis"
    Für die erste Anmeldung und das Laden neuer Befunde ist eine Internetverbindung erforderlich.
