# Support

Bei Fragen oder Problemen stehen wir Ihnen gerne zur Verfügung.

## Bevor Sie den Support kontaktieren

Prüfen Sie bitte zunächst:

1. **[FAQ](faq.md)** - Häufig gestellte Fragen
2. **Handbuch** - Durchsuchen Sie diese Dokumentation
3. **Neuigkeiten** - Gibt es bekannte Probleme?

## Support-Kanäle

### Labor-Support

Bei Fragen zu:

- Zugangsdaten
- Befundinhalten
- Laborspezifischen Themen

Kontaktieren Sie bitte direkt Ihr Labor. Die Kontaktdaten finden Sie unter "Labore" in der App.

## Fehlermeldung einreichen

Um uns bei der Problemlösung zu helfen, geben Sie bitte folgende Informationen an:

### Erforderliche Informationen

- [ ] **Gerät**: z.B. iPhone 14, Samsung Galaxy S23
- [ ] **Betriebssystem**: z.B. iOS 17, Android 14
- [ ] **Browser**: z.B. Chrome 120, Safari 17
- [ ] **App-Version**: Finden Sie unter Einstellungen → Über
- [ ] **Fehlerbeschreibung**: Was ist passiert?
- [ ] **Schritte zur Reproduktion**: Wie kann das Problem nachgestellt werden?

### Hilfreiche Zusatzinformationen

- Screenshots des Problems
- Fehlermeldungen (Text kopieren wenn möglich)
- Zeitpunkt des Auftretens
