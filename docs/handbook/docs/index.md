# Willkommen zum labGate PWA Handbuch

Willkommen beim offiziellen Benutzerhandbuch für die **labGate PWA** (Progressive Web App). Diese Dokumentation hilft Ihnen, alle Funktionen der Anwendung optimal zu nutzen.

---

## Was ist labGate PWA?

Die labGate PWA ist eine moderne Webanwendung, die Ärzten und medizinischem Fachpersonal einen schnellen und sicheren Zugriff auf Laborbefunde ermöglicht. Die App kann auf jedem Gerät mit einem modernen Webbrowser verwendet werden und bietet auch offline Zugriff auf bereits geladene Daten.

## Über dieses Handbuch

Dieses Handbuch ist im **PSR-Stil** (Problem Steps Recorder) aufgebaut:

- **Schritt-für-Schritt Anleitungen** mit nummerierten Schritten
- **Screenshots** zeigen jeden wichtigen Bildschirm
- **Benutzeraktionen** beschreiben, was Sie tun müssen
- **Ergebnisse** zeigen, was nach der Aktion passiert

!!! tip "Navigation"
    Nutzen Sie das Menü links, um zu verschiedenen Themen zu navigieren, oder die Suchfunktion oben rechts, um schnell bestimmte Themen zu finden.
