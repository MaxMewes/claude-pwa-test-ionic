# Benachrichtigungen

Konfigurieren Sie, wie und wann Sie über neue Befunde informiert werden.

## Push-Benachrichtigungen

### Aktivieren

1. Öffnen Sie "Einstellungen" → "Benachrichtigungen"
2. Aktivieren Sie "Push-Benachrichtigungen"
3. Erlauben Sie Benachrichtigungen, wenn vom Browser gefragt

!!! info "Browser-Berechtigung"
    Sie müssen der App die Berechtigung für Benachrichtigungen in Ihrem Browser erteilen.

### Deaktivieren

1. Öffnen Sie "Einstellungen" → "Benachrichtigungen"
2. Deaktivieren Sie "Push-Benachrichtigungen"

## Benachrichtigungsoptionen

### Neue Befunde
Erhalten Sie eine Benachrichtigung bei:

- [x] Alle neuen Befunde
- [x] Nur pathologische Befunde
- [x] Nur Notfallbefunde

### Neuigkeiten
- [x] Wichtige Ankündigungen
- [ ] Alle Neuigkeiten

### Stille Zeiten
Definieren Sie Zeiten, in denen keine Benachrichtigungen gesendet werden:

- **Von**: z.B. 22:00
- **Bis**: z.B. 07:00

## Benachrichtigungstöne

### Systemton
Verwendet den Standard-Benachrichtigungston Ihres Geräts.

### Vibration
Aktivieren/Deaktivieren Sie die Vibration bei Benachrichtigungen.

## Problembehandlung

### Benachrichtigungen kommen nicht an?

1. **Browser-Einstellungen prüfen**
   - Stellen Sie sicher, dass Benachrichtigungen für die App erlaubt sind

2. **Geräte-Einstellungen prüfen**
   - Nicht-Stören-Modus deaktivieren
   - App-Benachrichtigungen in den Systemeinstellungen erlauben

3. **App neu starten**
   - Schließen Sie die App vollständig
   - Öffnen Sie die App erneut

4. **Anmeldung prüfen**
   - Stellen Sie sicher, dass Sie angemeldet sind
   - Melden Sie sich ggf. ab und wieder an

### Test-Benachrichtigung

Tippen Sie auf "Test-Benachrichtigung senden", um die Funktion zu testen.

!!! tip "Tipp"
    Aktivieren Sie Benachrichtigungen besonders für pathologische Befunde, um keine wichtigen Ergebnisse zu verpassen.
